# CommitToEat 🍔

Сайт-помощник для студентов СибГУТИ.

## О проекте
CommitToEat помогает:
- Найти место, где можно успеть поесть во время большого перерыва
- Найти компанию для обеда (Обед-рулетка)

## Сборка и запуск
```bash
gcc -o main main.c
./main
```
